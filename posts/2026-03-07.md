(3.7.2026) I've really started to use claude code in earnest these last weeks. My initial thoughts are
<ul>
<li>the UX is really spectacular for surgical edits </li>
<li>It is still quite difficult to reach the level of precision to make more complicated apps. There is definitely "looking under the lamppost" syndrome. e.g. the danger of considering a number of implementations, mentally discarding the ones which you don't believe claude can code, and then trying out the rest. </li>
<li>It is much more frustrating to circle around something in plain language than it is in code. This is because not only do you need to find an inaccuracy in your own model, you need to successfully pick out the inaccuracy in the AI's model of the problem. You need to "talk up" to models in ways which allow it to make the right assumptions.</li>
</ul>

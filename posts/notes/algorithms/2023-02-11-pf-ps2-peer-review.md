Hi, this is a good try with some errors!

For problem 1(a), your set notation should be consistent: the 4th set should have brackets around it as $\{\emptyset, \{a\}\}$.

For problem 1(b), the cartesian product do not include ordered pairs where either element is empty. If you think of it in terms of (x,y) coordinates, it doesn't make sense to have something like (,2) or (2,) ever, right? We could have (1,2) or (0,2) but not (,2).

For 3.1, the intersection should be written $\emptyset$, not the set containing it; $\emptyset$ is not a member of either set being intersected.

For 4.2, consider that the union of intervals $[i, i+1]$ over all $i\in \mathbb{N}$ does not cover $\mathbb{R}$, only positive numbers!

For 4.3, I'm confused why you think it is a cuboid; where's the 3rd dimension?

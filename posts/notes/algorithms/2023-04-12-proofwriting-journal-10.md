The most important things I've learned is how to see proof structures and sets more viscerally. I heavily rely on visual intuition and while it is not essential for a proof, and it reinforces my ability to manipulate something when it is drawn in some form on the page. This might be because my spatial intuition is worse than many people, so that I need to actually see it to work with it. 

Overall, I feel comfortable with employing direct proof, contrapositive, and proof by contradiction. I think there will still be some challenge in knowing when to use each of these in new domains of math, but I think I will improve as time goes on. 

I have generally found feedback to be more helpful while I am still working on problem sets than after I have written it. This is because I can defend myself and realize my own mistakes. I have gotten somewhat contradictory responses from peer reviewers, but I still try to reread my proof and correct mistakes where possible. 

I think I have gotten a lot better at proofs by definition and built a lot more intuition around induction. I've been very happy with the speed at which I can typeset in LaTex; I write my statistics notes in LaTex during lecture now.

**Example running distribution on the address 127.0.0.1:**
```
distribution.local.routes.get('status', (e, v) => {
    v.get('sid', (e, result) => console.log('SID:', result));
    v.get('ip', (e, result) => console.log('IP:', result));
    v.get('heapUsed', (e, result) => console.log('Heap:', result));
    v.get('counts', (e, result) => console.log('Counts:', result));
});
```

Test for the 'NID' match with the comm send of the status.get with 
`{node: node, service: 'status', method: 'get'}`

```
distribution.local.comm.send(message, remote, (e, v) => {
  try {
    // Check that error is falsy
    if (e) {
      throw new Error(`Expected no error, but got: ${e}`);
    }
    
    // Check that value equals getNID result
    const expected = distribution.util.id.getNID(node);
    if (v !== expected) {
      throw new Error(`Expected ${expected}, but got ${v}`);
    }
    
    console.log('✓ Test passed');
  } catch (error) {
    console.error('✗ Test failed:', error.message);
    throw error; // Re-throw if you want the process to exit with error
  }
});
```

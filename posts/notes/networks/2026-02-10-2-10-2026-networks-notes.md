##### Internet Protocol
We create a network of networks, we use a router to forward packets between networks. This allows long-distance transmission of messages. 
- Router has information about other networks. Allows the 
- # internet


Packet switching:
- divide data into packets and move each packet separately
- for each packet, the router only needs to determine the next destination,no initial setup
- Packets may take any path
- Access is more random, so that it scales better vs. circuits

IMPs: early packet switched networks
==IP Decisions==

- Routers have no state, just need to forward the packet in the best possible manner. If a packet is dropped, whatever.

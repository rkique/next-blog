Wireshark

Will connect using Linux's loopback interface. Virtual network interface existing only in the local system. This is so that systems on the same network can communicate with networkign protocols even when there is no network!

Always has the IP address: `127.0.0.1`
Known as `localhost`: translated to IP address by the localhost.

`lo`: traffic to and from localhost.
`eth0`: link to the outside world, to see traffic leaving container.

**Handling a client on server side**
All information about client socket is in net.Conn returned by net.AcceptTCP.

How to deliver a package? Label with metadata
Layers of network: application, TCP / UDP, IP, Link Layer

Application protocol: Layer 7
What are examples of these protocols? They include HTTP/HTTPS, SSH, FTP, and DNS
- When building application, you use these protocols to communicate with one another and the world

Transport protocol: Layer 4
- These rely on the transport layer: the socket interface used by the OS
- Creates "pipes" for applications to send information back and forth from.
- Anatomy of a packet: 
	- Labeled with HTTP, TCP, and IP version numbers
	- Network layer (3): move packets between hosts anywhere on the Internet
	- routing and forwarding between nodes
- UDP: User Datagram Protocol
	- Sends packets to a port
	- packets may be dropped, reordered, duplicated
	- Faster than TCP: smaller headers, no connection setup

- TCP: Transmission Control Protocol
	- illusion of reliability
	- Can handle congestion and flow control
	- **Most programs use TCP**
		- Listen on well-known ports: 22, 25, 80, 443

TCP and UDP ports are independent (0 - 65535) = 2^16

`Network connection uniquely identified by IP address, transport protocol, and port number.`

Contain 16-bit fields for both the source and destination fieldds
Network protocol: Layer 3
- Applications are by and large built on IP.
- These are IPv4 and IPv6
Link-Layer protocol: Layer 2
- These are the moving of frames to other node via link
- Ethernet, Wifi (kB / s numbers)

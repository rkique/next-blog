We have a cubic press.

This cubic press has value only because of its ability to produce... ... which makes someone happy.
Therefore, the diamonds must have value.

We have penicillin. 

This penicillin has value only because of its ability to prevent ... ... which makes someone happy.
Therefore, the bacterial infections must have value!

We have ends.
These ends have value only because of their ability ... ... which makes someone happy (?)

??? Things are valuable only because we value them.
This could only be so if *we* have value.

Imagine a person who always says they value the thing for its own sake. 
There's just something arbitrary and dogmatic about valuing many such unrelated,
unsystematic, contingently-chosen ends, without some more fundamental 
explanation for why they matter.
